School Management System

The School Management System is a web application developed using CodeIgniter, a powerful PHP framework, designed to streamline the management operations of a school. This CRUD (Create, Read, Update, Delete) application provides essential functionalities for both teachers and students, enabling efficient management of academic information and resources.

Key Features

User Authentication

Teacher Dashboard

Student Dashboard

Course Management

Assignment Submission
