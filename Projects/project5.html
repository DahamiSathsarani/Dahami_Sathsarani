Lost Pet Finding App

Lost Pet Finder is a web platform developed using CodeIgniter, a PHP framework, aimed at helping users find their lost pets. This website provides a user-friendly interface for users to create accounts, publish lost pet articles, and connect with other users in the community to facilitate the safe return of lost pets.
